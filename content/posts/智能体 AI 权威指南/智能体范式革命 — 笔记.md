> “AI 的未来不仅是聊天机器人回答问题，而是智能体采取行动解决真实问题。”

## 本章概述

建立智能体的宏观认知框架。你将学习从静态 LLM 向动态自主智能体的演进，理解支撑智能体运行的核心理论模型，拆解四大核心组件，以及掌握从 L1 到 L5 的认知层级分类。

## 核心理论模型

| 模型 | 全称 | 用途 |
|------|------|------|
| **PEAS** | Performance, Environment, Actuators, Sensors | 智能体设计框架 |
| **POMDP** | 部分可观测马尔可夫决策过程 | 不确定性环境下的决策模型 |

## 四大核心组件

| 组件 | 作用 |
|------|------|
| **大脑** | 推理、决策、规划 |
| **感知** | 理解环境输入 |
| **行动** | 执行操作、影响环境 |
| **记忆** | 存储经验与上下文 |

## 认知层级（L1–L5）

| 层级 | 描述 |
|------|------|
| L1 | 基础响应 |
| L2 | 上下文理解 |
| L3 | 主动推理 |
| L4 | 自主规划 |
| L5 | 超级智能体 |

## 学习目标

1. **理解** 智能体与 LLM 的本质区别
2. **掌握** 智能体的核心理论模型与四大核心组件
3. **评估** 智能体的认知层级与能力边界
4. **应用** 智能体工作流范式优化系统设计

## 章节结构

范式转移 → 理论基础 → 核心组件 → 能力分层 → 工作流模式
